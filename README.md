# Library Management System

Console-based library management system written in C#.

This project is a capstone exercise focused on applying core software engineering concepts including:

- Object-Oriented Programming
- Inheritance and Polymorphism
- File Persistence
- Exception Handling
- Data Structures
- Git version control

## Features

- Manage library inventory
- Support multiple item types (Books, Audiobooks)
- Member tracking
- Loan and return system
- File-based persistence

## Architecture

The project is structured using a simple domain model approach.

Models:
- LibraryItem (abstract base class)
- PhysicalBook
- AudioBook
- ItemStatus enum

Future components will include:
- Member management
- Loan tracking
- File persistence
